# このアプリケーションの説明

webアプリケーションのURLやHTTPの意味合いを学ぶ為に作成されたアプリケーションのRuby実装です

# 利用しているライブラリ

* bundler
* sinatora

# インストール

```
gem install bundler
git clone https://github.com/shouhei/study-http
cd study-http
bundle install path --path=vendor/bundle
```

# 動かし方

```
ruby app.rb
```

# 動作確認

```
curl http://localhost:4567/
```
